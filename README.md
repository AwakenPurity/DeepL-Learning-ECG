# DeepL-Learning-ECG
基于深度学习的心律失常检测

所用数据集均来源于PhysioNet, 
网址：https://www.physionet.org/about/database/
> PhysioNet是一个研究资源，提供了广泛的生理信号数据库、软件和工具，用于生理数据的分析。它由麻省理工学院（MIT）的PhysioNet研究资源开发。PhysioNet托管了一系列免费的数据集，包括心电图（ECG）、脑电图（EEG）、血压记录和其他生理信号。
